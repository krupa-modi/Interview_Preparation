# 3. Difference Between SSR vs SSG vs CSR vs ISR 🔥🔥

| Feature        | SSR         | SSG         | CSR             | ISR                |
| -------------- | ----------- | ----------- | --------------- | ------------------ |
| Rendering      | Server      | Build time  | Browser         | Build + Background |
| Data Freshness | Real-time   | Static      | Client fetch    | Semi-fresh         |
| SEO            | Excellent ✅ | Excellent ✅ | Poor ❌          | Excellent ✅        |
| Speed          | Medium      | Fastest ⚡   | Slow first load | Fast               |
| Server Load    | High        | Low         | Low             | Low                |

---

# 🎯 Final Interview Answer

👉 SSR = fresh server-rendered pages
👉 SSG = static prebuilt pages
👉 CSR = rendering in browser
👉 ISR = static pages with automatic updates

👉 Choose based on:

* SEO
* Performance
* Data freshness

👉 “SSG gives best performance, SSR gives fresh data, CSR gives rich interactivity, and ISR balances both performance and freshness.” 🚀
